# kpf-media-stats

한국언론진흥재단 「언론 통계 자료 분석·활용 경진대회」 단독 참가 프로젝트.

## 개요
- 접수: daker.ai, 마감 **2026-07-23(목) 10:00**
- 심사: 1차 공개투표(8/3~8/12) → 2차 내부심사(8/13~8/21) → 발표 8/24
- 제출물: 자유양식 PDF 보고서
- 시상: 최우수 100만 / 우수 50만×2, 이사장상

## 주제 (확정)
**생산자–수용자 인식 격차.** 언론인조사와 언론수용자조사를 같은 축에 결합해
"기자가 중시하는 뉴스 vs 독자가 신뢰·소비하는 뉴스"의 간극을 정량화하고,
간극이 큰 지점에 **정책안 + 서비스안**을 제안한다.

차별점: ① 원자료(microdata) 직접 분석 ② 두 조사 결합(드묾) ③ 기자 경험 기반 해석.

## 폴더 구조
- `data/raw/` — KPF 원본 데이터 + 코드북 (git 미추적)
- `data/processed/` — 정제·결합 데이터 (git 미추적)
- `analysis/` — 분석 스크립트/노트북
- `report/` — 보고서 원고·자산, 최종 PDF
- `notes/` — 매핑 테이블·분석 노트 (작업 로그 본체는 옵시디언)
