## 메모
- 전 조사 raw가 csv/xlsx 동봉 → pandas로 바로 읽힘(.sav 라벨 필요시 pyreadstat).
- microdata는 .gitignore로 git 미추적. 결과·집계표만 공유.
- 각 폴더에 통계표·보고서 PDF 포함 → 빠른 수치 대조용.
