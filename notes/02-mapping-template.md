# 비교문항 매핑 테이블 (W1 핵심 산출물)

- 목적: 언론인조사 ↔ 수용자조사에서 **같은 축으로 비교 가능한 문항**을 짝짓는다.
이게 되면 "격차"를 정량화할 수 있고, 안 되면 주제 B를 접고 A로 선회한다.


| 개념(축) | 언론인조사 변수/문항 | 수용자조사 변수/문항 | 척도 일치? | 비교 방식 | 비고 |
|---|---|---|---|---|---|
| 언론의 역할·중요가치 | | | | 순위 역전 / 평균차 | 순위 역전 나오면 헤드라인감 |
| 뉴스 신뢰도 | (자기/직업 신뢰) | (언론 신뢰도) | | 격차지수 | |
| 저널리즘 품질(정확·공정·심층) | | | | 평균차 | |
| 플랫폼 인식(포털/유튜브) | | | | 분포 비교 | |
| 언론의 문제점 진단 | | | | 순위 비교 | 보통 간극 큼 |
| 생성형 AI 활용·신뢰 | | | | 분포 비교 | 문항 존재 여부 확인 |

## 격차 정량화 후보 지표
- **평균차**: 동일 5점척도 항목의 기자 평균 − 수용자 평균.
- **순위 역전**: 기자 상위 가치가 수용자 하위(또는 반대)인 항목 식별.
- **격차지수**: 항목별 |분포차| 합산 또는 정규화 지수(정의는 분석 단계에서 확정).
- **세그먼트**: 수용자 연령대별로 격차가 어떻게 달라지는가.

## go/no-go 기준
- 비교 가능 개념이 **최소 3개** 확보되면 B 진행.
- 3개 미만이면 A(뉴스소비 3개년 시계열)로 선회.
